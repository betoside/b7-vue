#53: Interpretado In-Browser vs Pré-compilado

    In-Browser
        tudo é feito no navegador do user.
        deixa o codigo totalmente visivel
        se o sistema for pesado pode travar no computer do user 
        para estudo é a melhor forma

    
    Pré-compilado
        como usar profissionalmente
        usando um gerenciador de pacote: npm ou yarn
        o inínio fica um pouco mais difícil
